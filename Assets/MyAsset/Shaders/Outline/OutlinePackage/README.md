# RenderGraph Outline (URP)

High quality outline effect for Unity URP based on RenderGraph.

---

## Requirements

- Unity 2022.3+  
- Universal Render Pipeline

---

## Installation

Install the package via **Package Manager** (Git URL or Local Path).

---

## Setup

1. Open **Universal Render Pipeline Asset**
2. Select your **Renderer (Forward / Forward+)**
3. In **Renderer Features** press **Add Renderer Feature**
4. Add **Outline Render Feature**
5. Save the asset

Done. The outline system is now active in your project.

---

## Usage

- Add a **Global Volume**
- Create / assign a **Volume Profile**
- Enable **Outline**
- Adjust thickness and color

---

## Notes

- Works with RenderGraph
- Compatible with Forward and Forward+
- Safe for builds and mobile
