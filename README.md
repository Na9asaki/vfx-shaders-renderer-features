# RenderGraph Outline — URP Renderer Feature

High quality outline effect based on Unity RenderGraph for Universal Render Pipeline.

---

## Installation (via Package Manager)

Unity allows installing packages directly from GitHub.

1. Open **Window → Package Manager**
2. Click the **+** button
3. Select **Add package from git URL...**
4. Paste this:
https://github.com/Na9asaki/vfx-shaders-renderer-features.git?path=/Assets/MyAsset/Shaders/Outline/OutlinePackage

5. Press **Add**

Unity will automatically download and install the package.

---

## Setup

1. Open your **Universal Render Pipeline Asset**
2. Select your active **Renderer (Forward / Forward+)**
3. In **Renderer Features** click **Add Renderer Feature**
4. Add **Outline Render Feature**
5. Save the asset

Done — the outline is now active.

---

## Requirements

- Unity 6.0+
- URP
